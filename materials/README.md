# 자료실 (materials)

학생 공개 자료실([library.html](../library.html))이 이 폴더를 실시간으로 읽어 자동 나열함.

## 사용법

- 이 폴더에 파일을 커밋하면 자료실 페이지에 **자동으로** 나타남. 별도 등록 작업 없음.
- 주제별로 묶으려면 하위 폴더를 만들면 됨. 예) `materials/지형/`, `materials/기후/`. 자료실에서 폴더로 표시되고, 누르면 안으로 들어감.
- 파일 이름이 그대로 화면에 표시되므로, 사람이 읽기 좋은 이름을 권장함. 예) `지형_1차시_판구조론.pdf`.

## 표시 방식

- **PDF**: 브라우저에서 바로 열림.
- **워드/PPT/엑셀** (`.docx` 등): Office 온라인 뷰어로 표시됨.
- **이미지**: 새 탭에서 열림.
- 그 외 형식(`.hwp`, `.zip` 등): 내려받기만 제공됨.

## 참고

- 이름이 `.`으로 시작하는 파일과 `README.md`는 목록에서 숨겨짐.
- GitHub Pages는 한 파일 100MB, 사이트 약 1GB 한도가 있음. 필기 스캔 PDF는 용량이 커질 수 있으니 유의 바람.
